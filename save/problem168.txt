问题编号:168


问题标题:lcy的环球之旅


问题描述:
　　在轻松斩获NOI金牌后，lcy到某市场占有率与腾讯持平的公司应聘，面试官抛出这样一个问题，一个环形道路上若干加油站，每个加油站可以供给一定油量，可以选择从环形轨道上任意一个加油站出发，并拥有该加油站可以供给的油量，之后每行进到一个加油站，都会耗费一定的油量。现在需要在环形轨道上行进一周，如果在路途中油量小于0，车就不能再继续前进了，面试官想让lcy找出是否能有一个点满足这样的情况，lcy非常轻松得解决了，顺利进入这家公司，赢娶白富美，走上人生巅峰。  
　　但是若干年后，lcy一家出去游玩时又遇到了此等问题，他们一家决定开飞机进行环球旅行，但是飞机的加油站非常有限，在旅途中只能遇见n个加油站，每个加油站能供给ai的油量，从第i个加油站飞到第i+1个加油站需要bi的油量，现在lcy决定从一个加油站出发，最终回到这个加油站，并在这个加油站所在的国家定居，于是他想知道有多少个加油站是可以作为出发点的。


输入格式:
第一行一个整数n，表示加油站个数  
第二行n个整数，表示每个加油站的供油量ai  
第三行n个整数，表示从第i个到第i+1个加油站需要耗费的油量bi(bn表示从第n个加油站到第1个加油站，)  


输出格式:
一行一个整数，表示可以选择的加油站个数  



测试样例:
####样例输入1

```
3
2 2 1
0 0 0
```
####样例输出1
```
3
```
####样例输入2
```
4
5 5 3 3
2 8 1 4
```
####样例输出2
```
2
```


数据范围与提示
####数据范围
1 &lt;= n &lt;= 2e6  
0 &lt;= ai &lt;= 1e3  
0&lt;= bi &lt;= 1e3 
####样例解释
```
对于第二个样例，我们有如下选择:
从1出发油量:5
到达2油量:8
到达3油量:3
到达4油量:5
返回1油量:1

从2出发油量:5
在未到达3的时候，油量已经变为-3，故不能从2出发

从3出发油量:3
到达4油量:5
到达1油量:6
到达2油量:9
返回3油量:1

从4出发油量:3
在未到达1的时候，油量已经变为-1，故不能从4出发
```
