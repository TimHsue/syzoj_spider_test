问题标题:圣诞节的连锁反应


问题描述:
###距离距离上一个圣诞节已经过去了半年的时间 ( 16 - 06 )
###然而这道题Chenyao还是没有造数据 --- By_Fmuckss  

圣诞节这天，一切变得有魔力起来。球掌握了黑魔法，可以在圣诞节这天隐藏其他的球。

有n个球排成一排。每个球都有自己的位置和破坏距离。当触发一个球的破坏行为时，球就会把自己**左边**的距离在自己破坏距离之内的球全部隐藏，当然球不会隐藏自己。现在你会对球序列按照从右到左的顺序触发破坏操作，但是如果一个球已经被隐藏，就不会触发它。

为了使得圣诞节的连锁反应破坏最小，你可以选在**所有球的右边**的**任意位置**放置一个**任意破坏距离**的球，得到一个新的球序列。接下来从新的球序列最右边开始不断触发破坏操作。现在请你找到一个放置新球的位置和破坏距离，使得最后被隐藏的球的数量尽量少。问最少多少的球被隐藏。


输入格式:
第一行一个整数n，表示球的数量。

接下来n行每行两个整数，分别表示一个球的位置和破坏距离。保证不会有两个球在一个位置上。


输出格式:
输出一个整数，表示最少使得几个球被隐藏。


测试样例:
## 样例输入一

```
4
1 9
3 1
6 1
7 4
```

## 样例输出一

```
1
```

## 样例输入二

```
7
1 1
2 1
3 1
4 1
5 1
6 1
7 1
```

## 样例输出二

```
3
```

## 样例解释
对于第一个样例，可以把球放到位置9，破坏距离为2。首先位置9的球把位置7的球隐藏起来，接下来位置6的球谁也隐藏不掉，位置3的球谁也隐藏不掉，位置1的球谁也隐藏不掉。共隐藏1一个球。

对于第二个样例，可以在位置2333放置一个破坏力为233的球。首先位置2333的球谁也隐藏不掉，接下来位置7的球隐藏掉位置6的球，位置5的球隐藏掉位置4的球，位置2的球隐藏掉位置1的球。共隐藏3个球。


数据范围与提示
1 &lt;= n &lt;= 100,000，保证其他输入数据均在`int`范围内。
