问题标题:[NOIP] 数字反转


问题描述:
[NOIP2011普] 数字反转
给定一个整数，请将该数各个位上数字反转得到一个新数。新数也应满足整数的常见形式，即除非给定的原数为零，否则反转后得到的新数的最高位数字不应为零（参见样例）。


输入格式:
输入共1 行，一个整数N。


输出格式:
输出共1 行，一个整数，表示反转后的新数。


测试样例:
####输入样例1
```
123
```
####输出样例1
```
321
```

####输入样例2
```
-380
```
####输出样例2
```
-83
```



数据范围与提示
-1,000,000,000 ≤ N≤ 1,000,000,000。
