问题标题:式量计算--[未审核]


问题描述:
窝来个水题吧。输入一个化学式，输出它的相对分子质量。不保证输入的物质真实存在。括号可能嵌套。不会出现配合物和•符号。化学式前没有系数。

```
元素名称	H	C	N	O	Na	Mg	Al	P	S	Cl	K	Ca
相对原子质量	1	12	14	16	23	24	30.5	16	32	35.5	19	20
```

输入格式:
物质的化学式，只由表中出现的元素构成


输出格式:
仅一行，式量，保留两位小数。


测试样例:
####输入样例0
```
CaCO3
```
####输出样例0
```
100.00
```
####输入样例1
```
Al(OH)3
```
####输出样例1
```
156.00
```
