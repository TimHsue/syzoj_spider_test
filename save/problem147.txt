问题标题:AC自动机练习


问题描述:
裸AC自动机  
给定一个字符串T，并给出一个字符串组S  
求S中的所有字符串在串T中出现的次数   


输入格式:
第一行一个字符串T T.length() &lt;= 1e7  
第二行一个整数n &lt;= 30，表示组S中字符串的个数  
接下来n行每行一个字符串，第i行表示组S中的字符串 Si Si.length() &lt;= 20  



输出格式:
按输入顺序依次输出组S中字符串及其在T中出现的次数  
中间用空格隔开


测试样例:
####样例输入
```
xttxttxtt
3
x
xt
t
```
####样例输出
```
x 3
xt 3
t 6
```


数据范围与提示
n &lt;= 30  
T.length() &lt;= 1e7  
Si.length() &lt;= 20
