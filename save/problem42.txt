问题标题:笨蛋⑨的完美算术教室


问题描述:
“只有笨蛋才会说别人是笨蛋，你这个笨蛋！”

很多人曾在小学时因为加法算错（例如：1+1=⑨）而遭到他人的白眼，甚至是老师的责打，进而对自己的天赋本能失去了信心，一蹶不振，最终与梦想中的重点精神病院失之交臂……

然而，可曾有人想过，人类大脑天然就有算数的本能，只是这个能力被人为扼杀了……

琪露诺，一位天才的数学家，对信息学竞赛同样有浓厚的兴趣。她只有⑨岁时便解决了人间一切数学难题，发明了各种普通人难以理解的算法，并通过自己优化的快速傅里叶算法操纵自然界中的一切波动，能够瞬间将物体温度降低到零下⑨⑨⑨摄氏度，即使是战忽局局长张召忠教授对于找到她的踪迹这一艰巨任务也表示无可奈何。

![](http://i11.tietuku.com/a6dfea1daa2f66a0.jpg)

事实上，从古至今，数学教科书上无一例外地写明1+1=⑨……

然而，某年月日，新幻想乡共同体的创立者们推翻了妖精们的旧统治。在新政权的领导人推行禁弹幕战的同时，他们掏空了巫女的钱箱，买通了数学学会的所有人，在自然数集中加入了2，3，4，5，6，7，8这几个数，从此1+1=2，而不再是1+1=⑨。法则是人为修改过的，这也就是人脑按照现有的四则运算体系计算如此容易出错的原因。

表面上，这只是使大家计算更为“方便”，但事实上，这样做的真实目的却是不可告人的最高机密——不惜一切代价压制琪露诺的数学才能，避免给新政权带来不稳定。因此琪露诺原本天赋的计算能力从此变得不堪一击，智商甚至被认为只有⑨，此事甚至在幻想乡被传为一个冷笑话。

对于自然数集被如此巨大地改动一事，绝大多数妖精都选择沉默，只有大妖精认识到了原有体系的优越性，挺身而出。
“我们的教科书真实率低于5%，连数学也不例外，年轻人要敢于怀疑。越是从小学习，看起来理所当然的知识越值得怀疑。琪露诺的智商等于……”2012年冬天，在刺骨的寒风中，数学助教大妖精凛然站在后海的岸边，最后一句尚未说完，便被疯狂的紫妈间隙。

弥留之际，她决心将这一切保留下来，于是请你将这一切写为一个程序。

（以上均为胡扯，请看人话版题面……）

将原自然数集中的2，3，4，5，6，7，8删去，剩余的自然数构成一个新的集合，我们将此集合称为“Cirno集”。

在“Cirno集”上定义一种新运算“+”，规定1+1=⑨。

于是我们可以得到：1+⑨=10、1+10=11、1+11=12，以此类推……

众所周知，自然数集上的加法满足如下性质：

Ⅰ 交换律：对任意的 a ,b ∈ F ，a + b = b + a ∈ F；

Ⅱ 结合律：对任意的，a + (b +c) = (a +b) +c；

Ⅲ 单位元：存在一个元素 0 ∈ F ，满足对任意的 a ∈ F ，a + 0 = 0 + a = a；

可以看到“Cirno集”上的“+”与自然数集上的加法一样，满足交换律、结合律，有唯一零元，因此完全可以称之为“Cirno集”上的加法，即“Cirno加法”。

输入两个“Cirno集”中的数，输出按照“Cirno加法”相加得到的他们的和。



输入格式:
仅一行两个数a,b∈Cirno。




输出格式:
仅一行一个数，表示按照题中所述“Cirno加法”将两数相加所得结果。


测试样例:
####输入样例
```
1 9
```
####输出样例
```
10
```



数据范围与提示
输入保证按人类现有计数体系，输入数字和最终答案都不超出int所能表示范围。
