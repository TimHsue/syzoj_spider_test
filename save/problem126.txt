问题编号:126


问题标题:[codeforces]Elephant 大象


问题描述:
（原题目见 http://codeforces.com/problemset/problem/617/A 超级大水题）（翻译by sxy）

大象打算去拜访他的朋友，他将自己的房子定位在位置0，将他朋友的房子定位在位置x (x &gt; 0)，他们的房子在一条直线上。大象每步可以移动1,2,3,4或5个位置，请问大象能到达他朋友家的最少步数是多少？


输入格式:
一行，一个整数x，表示它朋友家在位置x


输出格式:
一行，一个整数，表示大象到达他朋友家的最少步数


测试样例:
####样例输入1
```
5
```
####样例输出1
```
1
```
####样例解释1
```
#一步到达
```

####样例输入2
```
12
```
####样例输出2
```
3
```
####样例解释2
```
#他可以3,4,5 三步走完
```


数据范围与提示
x ∈ [1, 10^8]
