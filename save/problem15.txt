问题标题:[语言题]数组练习


问题描述:
有n个整数，分别为a(1),a(2),a(3)...a(n)。

现在询问你m次，第i次询问一个整数q(i)，问你a(q(i))是什么？


输入格式:
第一行两个整数，分别为n和m

第二行n个整数，分别为a(1),a(2),a(3)...a(n)。

接下来m行每行一个整数，分别为q(1),q(2)...q(m)。


输出格式:
输出m行，第i行为a(q(i))的值


测试样例:
####样例输入
```
5 3
50 13 45 60 79
3
1
5
```
####样例输出
```
45
50
79
```


数据范围与提示
1 &lt;= n,m &lt;= 100000

0 &lt;= a(i) &lt;= 2147483647

1 &lt;= q(i) &lt;= n
