问题标题:[AFO纪念版] 小试正则表达式


问题描述:
输入很多组字符串，每次判断这个字符串是否匹配这个正则表达式
```
ab((ba*)|(cd))ef*
```
*表示匹配前面的表达式1次或多次 ，|表示匹配两边的表达式的任意一个

匹配指的是这个字符串本身或者这个字符串内的一个连续子串能够被这个正则表达式接受

无聊水题


输入格式:
多组数据

第一行先输入一个整数T，表示数据组数

后面T行

每行一个字符串（不包含空格）（仅包含abcdef这几个字符）


输出格式:
对于每组数据，若匹配输出AC，否则输出WA


测试样例:
####样例输入
```
3
abcdef
abbaf
abbaaaefff
```
####样例输出
```
AC
WA
AC
```


数据范围与提示
AFO纪念版

字符串长度 len ∈ [1, 233333]

数据组数 T ∈ [1,233]
