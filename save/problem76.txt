问题标题: [Usaco2006 Oct]Another Cow Number Game 奶牛的数字游戏


问题描述:
奶牛们又在玩一种无聊的数字游戏。输得很郁闷的贝茜想请你写个程序来帮她在开局时预测结果。在游戏的开始，每头牛都会得到一个数$N(1 \le N \le 1,000,000)$。此时奶牛们的分数均为0。如果N是奇数，那么奶牛就会把它乘以3后再加1。如果N是偶数，那么这个数就会被除以2。数字每变动一次，这头奶牛就得到1分。当N的值等于1时，游戏结束，此时的分数就是这头奶牛在这局游戏中的最终得分。 


输入格式:
* 第1行: 一个正整数，N


输出格式:
* 第1行: 输出一个正整数N，即奶牛在这局游戏中的最终得分


测试样例:
####输入样例
```
112
```
####输出样例
```
20
```


数据范围与提示
$1 \le N \le 1,000,000$
