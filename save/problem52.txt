问题编号:52


问题标题:[noip2004]不爽的毛毛


问题描述:
快递员毛毛上班了。作为快递点的老板的你负责为毛毛派单，每天只派两单，但完成每单耗费时间有所不同。毛毛如果一天工作时间累计超过八个小时就会不爽，而且工作时间越长就会越不高兴。假设毛毛不会因为其它事不高兴，并且她的不高兴不会持续到第二天。请你帮忙检查一下毛毛下周的派单安排，看看下周她会不会不爽；如果会的话，哪天最不爽。


输入格式:
输入包括七行数据，分别表示周一到周日的日程安排。每行包括两个小于10的非负整数，用空格隔开，分别表示两单所耗时间。


输出格式:
一行，这一行只包含一个数字。如果不会不爽则输出0，如果会则输出最不爽的是周几（用1, 2, 3, 4, 5, 6, 7分别表示周一，周二，周三，周四，周五，周六，周日）。如果有两天或两天以上不爽的程度相当，则输出时间最靠前的一天。


测试样例:
输入
```
4 2
1 8
3 6
4 0
7 0
0 0
0 5
```
输出
```
2
```


数据范围与提示
[0,10)
