问题标题:[语言题+]求a除以b的高精度值


问题描述:
输入两个整数，求a除以b高精度值，保留20位小数


输入格式:
两个空格隔开的整数


输出格式:
一行，精确到小数点后20位


测试样例:
输入
```
6  5  
```
输出
```
6/5=1.2
```
输入
```
5  3  
```
输出
```
5/3=1.66666666666666666666
```
输入
```
1034 1033
```
输出
```
1034/1033=1.000968054211035818
```


数据范围与提示
考虑手算除法时，你是如何处理的，模拟手算即可。注意末尾零的处理
