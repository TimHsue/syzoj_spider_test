问题编号:113


问题标题:[H2O]资源合理分配


问题描述:
在多线程的环境中，多个线程可能会为同一个系统资源而产生冲突。比如A希望现在使用打印机设备10s，但是2s后B要使用打印机3s，但打印机只有1台。因此，我们需要协调各个线程之间的冲突。

现在我们的电脑上有N个线程，他们都想要使用你的打印机资源。你只有一台打印机。所以，操作系统决定，对所有的申请使用打印机的线程，采取先来后到的策略。即A先到，要求使用10s，A先使用。B后到，**哪怕B只要求使用1s，操作系统也不会打断A的工作去给B用。**直到A使用完毕，才开放给B使用。如果在B等待A的过程中，又来个C要求使用该资源，那么C会排队排到B的后边。 以此类推。


输入格式:
第一行一个整数N，表示现在有N个线程，希望在现在，或者将来的某个时间段使用你的打印机。

下面共有N行输入。第i行上如表示第i个线程的需求。每行2个数:a,b。表示这个线程要在时间段[a,b]内使用打印机


输出格式:
一个整数T，表示完成这N个线程的任务，所需的时间。


测试样例:
####样例输入
```
2
1 10
2 3
```

####样例输出
```
12
```


数据范围与提示
####友情提示
**读题能力是很重要的**

####数据范围
```
1 &lt;= N &lt;= 1000
1 &lt;= a,b &lt;= 100
```
