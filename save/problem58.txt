问题标题:[语言题]判断质数


问题描述:
输入一个质数，现在请你判断是否是质数。如果是的话输出&#34;Yes&#34;否则输出&#34;No&#34;。


输入格式:
输入有多组数据，第一行为一个整数Q，表示有Q组数据。

接下来有Q行，每行一个整数，为待判断的数字。


输出格式:
输出Q行，每行输出Yes或者No，表示是否是质数。注意大小写。


测试样例:
样例输入一
```
3
5
6
7
```
样例输出一
```
Yes
No
Yes
```

样例输入二
```
2
4
9
```
样例输出二
```
No
No
```


数据范围与提示
对于80%的数据，2 &lt;= 需要判断的数 &lt;= 10^5

对于100%的数据，1 &lt;= Q &lt;= 10, 2 &lt;= 需要判断的数 &lt;= 10^8
